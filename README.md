# 🚴 Bike Purchase Behavior Analysis

## 📌 Project Overview
This project analyzes customer demographics and purchasing behavior to identify key factors influencing bike purchase decisions. The goal is to derive actionable insights that can help businesses improve customer targeting and marketing strategies.

---

## 🎯 Objective
To explore customer data and understand how factors such as age, income, occupation, and commute distance impact bike purchasing behavior.

---

## 🛠️ Tools & Technologies
- Python  
- Pandas  
- Matplotlib  
- Seaborn  

---

## 📊 Dataset Information
- **Source:** Kaggle  
- **Dataset Name:** Bike Buyers Dataset  
- **Format:** CSV (Structured Data)  


---

## 🔧 Steps Performed
1. Data Collection from Kaggle  
2. Data Cleaning (handling missing values, formatting)  
3. Exploratory Data Analysis (EDA)  
4. Data Visualization using charts and graphs  
5. Insight Generation  

---

## 🔍 Key Insights
- Middle-aged individuals are more likely to purchase bikes  
- Higher income increases the likelihood of buying a bike  
- Commute distance significantly influences purchase decisions  
- Family size also impacts buying behavior
